#### What are python decorators ?
- decorators are the annotations which we have add on top of function which starts with '@'. Why needed ? A predefined method, just by using its name with '@', will make the function work accordingly. for eg: if get protocol called then this method should be used like that.

#### `async def` means ?
- if any function in python called with `async` then the function will work asynchronous manner i.e. hit function and complete execution and move foreward, once we get the result, again come back to the function and complete the function and remove from stack. If not `async` then the function will wait till it get the result.
